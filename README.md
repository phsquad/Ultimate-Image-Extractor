
# 🖼️ Ultimate Media Extractor Pro (Multi-Core)

[![Python Version](https://img.shields.io/badge/python-3.10%2B-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-lightgrey.svg)]()

**Ultimate Media Extractor Pro** — это высокопроизводительный инструмент на базе Python, предназначенный для глубокого анализа файлов и извлечения изображений из любых цифровых контейнеров. Программа использует мощности многоядерных процессоров и технологии системного уровня для поиска медиаданных там, где обычные просмоторщики бессильны.

---

## 🚀 Зачем и для чего создан этот проект?

Часто данные хранятся не в виде обычных картинок, а "спрятаны" внутри сложных структур. Этот софт был создан для решения следующих задач:

1.  **Восстановление данных из кэша**: Извлечение изображений из кэша браузеров, мессенджеров (Telegram, WhatsApp) и игр, где файлы часто не имеют расширений.
2.  **Парсинг дампов и экспортов**: Быстрая выгрузка медиа из гигантских JSON-файлов (например, экспорт истории Telegram) или HTML-страниц.
3.  **Цифровая археология (Carving)**: Поиск "цифровых отпечатков" изображений внутри поврежденных файлов, бинарных дампов или баз данных SQLite.
4.  **Автоматизация**: Когда нужно собрать тысячи фото из сотен разных папок, архивов и документов в одно место за один клик.

---

## ✨ Ключевые особенности

*   **⚡ Многоядерная обработка (Multi-Threading)**: Распараллеливает задачи на все доступные ядра вашего CPU, что позволяет обрабатывать тысячи файлов одновременно.
*   **🧠 Memory Mapping (mmap)**: Технология прямого отображения файлов в память. Позволяет мгновенно сканировать файлы размером 10ГБ+ без зависания системы и переполнения ОЗУ.
*   **🔍 Глубокий бинарный поиск (Carving)**: Находит изображения по "магическим байтам" (сигнатурам) внутри любых бинарных данных.
*   **📦 Поддержка форматов**:
    *   **Архивы**: ZIP, TAR, GZ, APK, DOCX, XLSX, EPUB.
    *   **Базы данных**: SQLite / DB (извлечение BLOB-объектов).
    *   **Документы**: PDF (через PyMuPDF).
    *   **Код**: Извлечение Base64-кодированных строк из JSON, HTML, CSS, JS.
*   **🌐 Сетевой модуль**: Автоматическое скачивание оригиналов изображений по найденным в коде ссылкам.
*   **🛑 Безопасная остановка**: Кнопка "Стоп" позволяет прервать процесс в любой момент без риска повреждения данных.

## 📥 Установка и запуск

### 🔹 Windows

1.  **Установите Python**: Скачайте и установите [Python 3.10+](https://www.python.org/downloads/) (обязательно отметьте галочку **"Add Python to PATH"**).
2.  **Скачайте проект**:
    ```bash
    git clone https://github.com/ВАШ_НИК/Ultimate-Image-Extractor.git
    cd Ultimate-Image-Extractor
    ```
3.  **Установите зависимости**:
    ```bash
    pip install -r requirements.txt
    ```
4.  **Запустите**:
    ```bash
    python app.py
    ```

### 🔹 Linux (Ubuntu/Debian/Fedora)

1.  **Установите системные зависимости**:
    ```bash
    sudo apt update
    sudo apt install python3-pip python3-tk
    ```
2.  **Склонируйте и установите**:
    ```bash
    git clone https://github.com/ВАШ_НИК/Ultimate-Image-Extractor.git
    cd Ultimate-Image-Extractor
    pip install -r requirements.txt
    ```
3.  **Запустите**:
    ```bash
    python3 app.py
    ```

## 📦 Сборка в EXE (для Windows)

Если вы хотите создать переносимый файл, который работает без установленного Python:

1.  Установите PyInstaller: `pip install pyinstaller`
2.  Запустите сборку:
    ```bash
    pyinstaller --noconsole --onefile --collect-all customtkinter --icon=icon.ico app.py
    ```
3.  Заберите готовый файл в папке `dist/`.

## 📖 Как пользоваться

1.  **Источник (Source)**: Выберите папку, которую нужно просканировать, или конкретный файл (например, `attachments.json`).
2.  **Папка сохранения (Destination)**: Выберите пустую папку, куда программа сложит все найденные фото.
3.  **Настройки**:
    *   Включите **"Network"**, если хотите докачивать фото по ссылкам из интернета.
    *   Включите **"Carving"**, если нужно искать скрытые фото в бинарных файлах.
4.  **Старт**: Нажмите кнопку запуска и наблюдайте за процессом в реальном времени через встроенную консоль логов.
