# 🚀 Ultimate Image Extractor (Multi-Core)

Мощная утилита на Python для глубокого извлечения изображений из любых файлов: HTML, JSON, PDF, SQLite, архивов и даже бинарных данных (кэш).

## ✨ Особенности
- **Multi-Core Processing**: Использует все ядра вашего процессора для параллельной обработки.
- **Memory Mapping (mmap)**: Мгновенный поиск в гигабайтных файлах без нагрузки на ОЗУ.
- **Smart Carving**: Находит скрытые JPEG/PNG по сигнатурам внутри поврежденных или неизвестных файлов.
- **Deep Parsing**: Извлекает Base64 из кода и скачивает изображения по прямым ссылкам.
- **Modern UI**: Красивый интерфейс с темной темой и логами в реальном времени.
